## Address Lookup
You can use [Nominatim](https://nominatim.openstreetmap.org/) to look up latitude and longitude for an address.

Example query for "Bonn":
```
https://nominatim.openstreetmap.org/search?limit=1&format=json&layer=address&q=Bonn
```
